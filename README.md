# Nahidur Zaman Tushar — Portfolio

A clean, modern, and highly responsive developer portfolio website built with **React**, **Vite**, and **Tailwind CSS**, powered by **Supabase** for backend operations, data management, and authentication.

---

## 🚀 Features

*   **Dynamic Data:** Project showcases, work experience, education, and skill metrics are directly fetched from a Supabase database.
*   **Fully Responsive:** Designed using Tailwind CSS with a mobile-first approach.
*   **Modern UI:** Features animations, icon integration using `lucide-react`, and elegant section transitions.
*   **SQL Schema Ready:** Pre-configured database schema for hassle-free deployment.

---

## 🛠️ Tech Stack

*   **Frontend:** React 18 + Vite + Tailwind CSS
*   **Icons:** Lucide React
*   **Backend & Database:** Supabase (PostgreSQL)
*   **Deployment:** Vercel
